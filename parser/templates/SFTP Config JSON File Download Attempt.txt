===============================

Mitigasi :<br>
- Hapus file sftp-config.json atau file konfigurasi sensitif dari direktori publik/webroot.
- Rotasi seluruh kredensial SFTP yang tersimpan di file tersebut.
- Terapkan key-based authentication dengan passphrase, bukan password plaintext.
- Batasi akses SFTP hanya dari IP yang terpercaya (whitelisting firewall/ACL).
- Terapkan file permission ketat agar konfigurasi tidak dapat diakses publik.
- Terapkan mekanisme secure configuration management (jangan simpan credential di file proyek).
